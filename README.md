# MERN_STACK
This is a simple trading bot which generates live market data then use that same data in a simple sell strategy and finally show everything(profit &amp; loss) on screen. 

# Installation
Install the dependencies
```sh
$ npm install
```
Start server
```sh
$ npm start
```
